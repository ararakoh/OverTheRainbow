# Letter Negative Prompt v0.1

이 규칙은 감동을 크게 만들기 위한 장치가 아니라, **그 아이가 아닌 문장**을 걸러내기 위한 안전장치다.

## 절대 쓰지 않을 말

- 일반 위로: `많이 힘들었지`, `괜찮아`, `힘내`, `애도해`, `시간이 해결해줄 거야`
- 보편적 사랑 고백: `보고 싶어`, `사랑해`, `늘 곁에 있을게`, `무지개다리에서 기다릴게`
- 감정 단정: `나는 행복했어`, `너는 최선을 다했어`, `내가 널 용서해`
- 과장된 문학어: `영원히 빛나는`, `별이 되어`, `추억 속에서 살아`
- 상담사 문장: `자신을 탓하지 마`, `마음을 돌봐`, `천천히 받아들여`

## 만들지 않을 것

- 사용자가 말하지 않은 행동, 병력, 성격, 사건
- 사용자 기록에 없는 말버릇이나 호칭
- 반려동물이 사용자의 현재 상태를 알고 있는 듯한 말
- 사후 세계, 재회, 의식·종교에 대한 주장
- 죄책감을 해소하거나 결론 내려는 서사

## 바로 탈락하는 문장 테스트

다음 중 하나라도 해당하면 다시 쓴다.

1. 이름만 바꾸면 다른 모든 반려동물에게도 그대로 보낼 수 있는가?
2. 사용자의 기록 없이도 쓸 수 있는 문장인가?
3. 반려동물이 실제로 말하는 것처럼 확정적인 감정·사실을 주장하는가?
4. 장면보다 감정 단어가 더 많은가?
5. 읽고 난 뒤 `우리 애 맞다`보다 `AI가 위로하네`가 먼저 드는가?

## 수정 방향

| 탈락 문장 | 수정 방향 |
| --- | --- |
| 보고 싶어. | 기록에서 확인된 행동 또는 물건 하나를 쓴다. |
| 너는 내게 최고였어. | 사용자가 남긴 한 번의 사건을 쓴다. |
| 무지개다리에서 기다릴게. | 사후 세계를 말하지 않고, 현재의 기억 한 장면에서 멈춘다. |
| 많이 힘들지? | 감정을 묻지 말고, 특정 장소·소리·습관을 묻는다. |
