# Memory Cue Library v0.1

이 문서는 예쁜 문장을 모으는 곳이 아니다. **사용자가 자기 반려동물의 장면을 꺼낼 수 있는 단서**를 모으는 곳이다.

## 사용 공식

`생활 단서` → `한 가지 질문` → `중립적인 행동 버튼`

예: `오늘은 비가 오는 날이에요.` → `이런 날, [이름]은 어느 쪽에 있었나요?` → `생각나는 곳 적기`

## 우선 수집할 재료

1. **반려동물 추모글의 공개된 관찰 문장**: 감정 표현을 인용하거나 복제하지 않고, 무엇을 기억하는지의 범주만 추출한다.
2. **사용자가 남긴 사진·기록·답변**: 이후에는 일반 질문보다 이 데이터에서 나온 장면을 우선한다.
3. **사별 기록과 에세이**: Joan Didion, C.S. Lewis는 감정을 설명하기보다 행동과 생각의 흐름을 보여주는 방식만 참고한다.
4. **짧은 대화와 인터랙션**: *Spiritfarer*, *To the Moon*, *A Short Hike*, *Animal Crossing*은 일상과 기억을 연결하는 장면 전환을 참고한다.
5. **한국어 생활 관찰문**: 김소영, 김금희, 정혜윤의 문장을 그대로 가져오기보다 생활 장면을 과장 없이 보여주는 태도를 참고한다.

## 단서 카테고리

### 1. 장소

- 창가
- 현관 앞
- 소파 모서리
- 이불 안
- 식탁 밑
- 자주 가던 산책길

질문 예시:

- `집에서 [이름]이 제일 오래 머물던 곳이 있었나요?`
- `그 자리를 보면 지금도 먼저 떠오르는 장면이 있나요?`

### 2. 날씨와 시간

- 비가 오는 날
- 햇볕이 드는 오후
- 더운 바닥
- 바람 부는 날
- 새벽
- 퇴근 시간

질문 예시:

- `오늘은 [날씨]한 날이에요. 이런 날, [이름]은 보통 어디에 있었나요?`
- `아침에 가장 먼저 하던 일이 있었나요?`

### 3. 소리와 신호

- 사료 봉지 소리
- 간식 봉지 소리
- 현관문 여는 소리
- 열쇠 소리
- 가방 지퍼 소리
- 이름을 부르는 소리

질문 예시:

- `간식 봉지 소리가 나면 [이름]은 어떻게 했나요?`
- `현관문 소리를 들으면 제일 먼저 보이던 모습이 있었나요?`

### 4. 몸짓과 습관

- 귀가 먼저 움직임
- 꼬리를 흔듦
- 기지개를 켬
- 발밑을 맴돎
- 장난감을 물고 옴
- 특정 자세로 잠듦

질문 예시:

- `이름을 부르면 어떤 표정이 떠오르세요?`
- `[이름]만 하던 버릇이 하나 있었나요?`

### 5. 함께한 루틴

- 산책 준비
- 밥그릇 채우기
- 퇴근 후 맞이하기
- 이불 속으로 들어오기
- 새벽에 깨우기
- 병원 다녀온 뒤 조용히 잠들기

질문 예시:

- `산책 나가기 전에는 늘 어떤 순서가 있었나요?`
- `집에 들어왔을 때 [이름]은 보통 어디에 있었나요?`

### 6. 물건

- 목줄
- 낡은 장난감
- 밥그릇
- 담요
- 이동장
- 사진 속 배경

질문 예시:

- `아직도 눈에 들어오는 물건이 있나요?`
- `그 물건을 보면 어떤 장면이 먼저 떠오르세요?`

## 민감한 장면의 사용 원칙

- 병원, 마지막 날, 빈자리처럼 강도가 높은 단서는 초기 온보딩의 기본 질문으로 쓰지 않는다.
- 사용자가 먼저 기록했거나 선택한 경우에만 관련 후속 질문을 제안한다.
- 질문이 기억을 단정하면 안 된다. `병원 가던 날이 떠오르죠?`가 아니라 `떠오르는 날이 있다면, 적어둘 수 있어요.`

## 카피 작성 금지선

- 단서 없이 감정을 단정하지 않는다. `많이 힘드셨죠` 금지.
- 반려동물의 마음을 대신 말하지 않는다. `[이름]도 보고 싶어 할 거예요` 금지.
- 답변을 요구하지 않는다. `기억을 남겨요` 대신 장면을 보여주고, 행동은 버튼으로만 둔다.
- 사용자 데이터가 없을 때 특정 행동을 사실처럼 말하지 않는다. `창밖을 좋아했죠`가 아니라 `창밖을 자주 보던 아이였나요?`

## MVP용 첫 6개 프롬프트

1. `가장 먼저 떠오르는 곳이 있나요?` — 창가, 산책길, 현관 앞, 자주 눕던 자리.
2. `이름을 부르면 어떤 표정이 떠오르세요?`
3. `오늘은 [날씨]한 날이에요. 이런 날, [이름]은 보통 어디에 있었나요?`
4. `간식 봉지 소리가 나면 [이름]은 어떻게 했나요?`
5. `집에 들어왔을 때 제일 먼저 보이던 모습이 있었나요?`
6. `[이름]만 하던 버릇이 하나 있었나요?`

각 프롬프트는 기록·건너뛰기·다음에 보기의 반응률과 후속 기록 길이를 비교해 교체한다. 좋은 카피는 문학적으로 아름다운 문장이 아니라, 사용자가 구체적인 장면 하나를 실제로 적게 만드는 문장이다.
